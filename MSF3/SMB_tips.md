##  Додаткові налаштування Metasploitable3 (якщо завантажили за інструцією)

Щоб ви могли успішно пройти цей етап завдання, зробіть наступне у терміналі Metasploitable3:

**1. "Оживіть" користувача для лаби:**
```bash
# Змінюємо системний пароль
sudo passwd chewbacca
# (wookiee або будь який інший)

# Оновлюємо пароль у базі Samba (це критично для Metasploit)
sudo smbpasswd -a chewbacca
# (wookiee або будь який інший, бажано сінхронізувати (системний та для SMB))
```

**2. Перевірте результат у Kali:**
Тепер запустіть свій `smb_login` у Metasploit знову. Він спрацює миттєво.
---

## **"Важливе зауваження щодо стенда"**:

> "У процесі розгортання Metasploitable3 може виникнути розсинхронізація паролів. Якщо під час розвідки (`nmap`) ви бачите користувача, але брутфорс не дає результатів, це може свідчити про унікальний хеш пароля, згенерований під час інсталяції. Для цілей навчання адміністратор стенда може синхронізувати паролі вручну, використовуючи команди `passwd` та `smbpasswd`".
