# Перелік та детальний опис вразливостей --- Vikings 1

## 1. Відкрита мережна поверхня атаки (SSH та HTTP)

**Опис:**\
Під час сканування було виявлено відкриті порти SSH (22) та HTTP (80).
Сервіси доступні без додаткових обмежень доступу.

**Ризик:**\
- Можливість брутфорсу облікових даних. - Розвідка сервісів та пошук
вразливостей. - Збільшення поверхні атаки.

**Наслідки:**\
Initial Access, Information Disclosure.

**Рекомендації:**\
- Обмежити доступ по IP. - Використовувати firewall. - Вимкнути
непотрібні сервіси.

------------------------------------------------------------------------

## 2. Directory Enumeration (Gobuster)

**Опис:**\
Вебсервер дозволяє перебір директорій та файлів без захисту, що дозволяє
виявити приховані ресурси.

**Ризик:**\
- Витік конфігураційних файлів. - Отримання службових файлів.

**Наслідки:**\
Sensitive Data Exposure.

**Рекомендації:**\
- Заборонити directory listing. - Використовувати WAF. - Перевіряти
права доступу.

------------------------------------------------------------------------

## 3. Незахищене зберігання архівів із слабким паролем

**Опис:**\
ZIP-архів захищений слабким паролем, який був підібраний словником.

**Ризик:**\
- Компрометація даних. - Подальший розвиток атаки.

**Рекомендації:**\
- Використовувати складні паролі. - Шифрувати чутливі дані.

------------------------------------------------------------------------

## 4. Приховані дані в мультимедіа (Steganography)

**Опис:**\
Виявлено приховану інформацію в зображенні.

**Ризик:**\
- Прихований канал передачі даних. - Обхід систем контролю.

**Рекомендації:**\
- Контроль файлів. - Антивірусний аналіз.

------------------------------------------------------------------------

## 5. Слабка аутентифікація користувача

**Опис:**\
Отримано пароль користувача шляхом аналізу.

**Ризик:**\
- Несанкціонований доступ. - Перехід між користувачами.

**Рекомендації:**\
- MFA. - Політика складних паролів.

------------------------------------------------------------------------

## 6. Небезпечний RPC-сервіс rpyc_classic

**Опис:**\
Сервіс дозволяє виконання довільного коду з правами root.

**Ризик:**\
- Повна компрометація системи.

**Рекомендації:**\
- Вимкнути сервіс. - Обмежити доступ. - Оновити бібліотеки.

------------------------------------------------------------------------

## 7. Неправильна конфігурація автозапуску (.profile)

**Опис:**\
Небезпечний сервіс запускається автоматично.

**Ризик:**\
- Персистентність атаки.

**Рекомендації:**\
- Перевіряти автозапуск. - Мінімізувати привілеї.
