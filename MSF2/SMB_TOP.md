Порт **445 (Microsoft-DS / SMB over TCP)** стабільно посідає перші місця в антирейтингах безпеки протягом останніх десятиліть. Це пояснюється тим, що SMB (Server Message Block) є критично важливим для роботи мереж Windows, але водночас він має довгу історію критичних вразливостей.

Ось головні докази та статистичні дані з авторитетних джерел:

### 1. Статистика сканування та активності ботнетів (SANS ISC)

Проєкт **SANS Institute Internet Storm Center (ISC)** веде безперервний моніторинг глобальної активності портів. Порт 445 майже завжди входить до **ТОП-3** найбільш сканованих портів у всьому світі.

  * **Доказ:** Згідно з даними [SANS ISC «Top Reports»](https://isc.sans.edu/top10.html), порт 445 постійно демонструє аномально високу кількість «target IP» та «reports». Це означає, що тисячі автоматизованих скриптів та ботнетів щосекунди шукають відкриті SMB-сервери для спроб брутфорсу або експлуатації.

### 2. Звіт Rapid7: «Project Sonar» та аналіз експозиції

Компанія **Rapid7** (розробник Metasploit) регулярно публікує *Industry Cyber-Exposure Report*.

  * **Статистика:** В одному з масштабних досліджень було виявлено, що понад **1.6 мільйона** пристроїв в інтернеті мають відритий порт 445 із підтримкою вразливого протоколу SMBv1.
  * **Чому це важливо:** Це робить порт 445 «найкоротшим шляхом» для черв'яків (ransomware worms).
  
### 3. Спадщина EternalBlue та WannaCry

Найбільш руйнівні кібератаки в історії базувалися саме на експлуатації порту 445.

  * **Факт:** Атака **WannaCry (2017)** вразила понад 200,000 систем у 150 країнах за лічені години, використовуючи експлейт **EternalBlue** (CVE-2017-0144) через порт 445.

### 4. Звіт Palo Alto Networks

Дослідницька група **Unit 42** регулярно зазначає, що SMB є найпопулярнішим вектором для латерального переміщення (lateral movement) всередині мережі.

  * **Статистика:** У звітах про загрози хмарним та корпоративним мережам, експлуатація слабких паролів SMB або вразливостей (наприклад, **SMBGhost** — CVE-2020-0796) фігурує як основний метод закріплення зловмисників.

### 5. Пошукові системи (Shodan)

Якщо заглянути в [Shodan](https://www.shodan.io), можна побачити мільйони відкритих портів 445. Велика кількість цих вузлів належить до критичної інфраструктури або застарілих серверів, що робить їх «ласим шматком» для атак.

-----

### Чому саме 445? (Підсумок для Metasploit)

Для користувача Metasploit порт 445 — це «золотий стандарт», тому що:

1.  **RCE (Remote Code Execution):** Багато вразливостей SMB дають права системи (`NT AUTHORITY\SYSTEM`).
2.  **Information Gathering:** Через SMB можна отримати список користувачів, спільних папок та версію ОС без автентифікації (у застарілих системах).
3.  **Credential Hunting:** Протокол дозволяє проводити атаки типу SMB Relay або брутфорс паролів.
